Creating an array

1. Using an array literal is the easiest way to creat an array in Javascript.

Code:

  function declareArray() {
                const cars = ["Saab", "Volvo", "BMW"]

2. You can Also create an array, and then provide the elements.

Code:

    function createAndProvide() {
        const cars = [];
        cars[0] = "Saab";
        cars[1] = "Volvo";
        cars[2] = "BMW";
    }

Reversing An Array

reverse() method reverses the elements in an array. You can use it to sort an array in descending order.

Code:

    function reverseArray() {
        const fruits = ["Banana", "Orange", "Apple", "Mango"];
        fruits.sort();
        document.getElementById("revArray").innerHTML = fruits.reverse();
    }

Numeric Sorting an Array

By Default, the sort() function sorts values as strings.

This works well for strings ("apple" comes before "banana").

However, if numbers are sorted as strings, "25" is bigger than "100", because "2" is bigger than "1".

Because of this, the sort() method will produce incorrect result when sorting numbers.

You can fix this by providing a compare function

Code:

    function numericSort() {
        const points = [40, 100, 1, 5, 25, 10];
        points.sort(function (a, b) { return a - b });
    }
