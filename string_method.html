3. slice() extracts a part of a string and returns the extracted part in a new string. The method takes 2 parameters: the start position, and the end position (end not included). This example slices out a portion of a string from position 7 to position 12 (13-1):

Code:

            function slicing() {
              let str = "Apple, Banana, Kiwi";
          
              document.getElementById("strslice").innerHTML = str.slice(7, 13);
            }
